## 第七章：社区角色与参与体系——多元参与，共同成长

我们为每一位共创伙伴设计了清晰、多元的参与路径，确保每一份贡献都被看见，每个人都能找到适合自己的位置。

我们把完成认证、愿意和社区一起共建项目的成员称为“共创伙伴”；而刚进入社区、正在了解的朋友，会先加入开放社群（开放群）进行体验。

> 伙伴 = 深度共创的朋友，一起创造，不是付费会员

### 7.1 参与层级：从新朋友到共创伙伴

**新朋友**
- 对社区感兴趣、想先体验的朋友
- 通过开放群获取活动信息，随时旁听或报名
- 参与公开活动，感受社区节奏，寻找想加入的项目

**共创伙伴 (Co-creation Partner)**
- 完成认证的成员
- 加入共创伙伴核心群，参与深度解读、资源对接、项目协作
- 认证条件：完善网站档案 + 一次有意义的贡献（志愿服务、分享、黑客松参赛等）
- 邀请制：至少有一名伙伴邀请，需要填写邀请人信息

### 7.2 共创伙伴能解锁什么
- 和一群靠谱的创造者持续深聊，找到项目同伴、创业合伙人
- 共同参加黑客松、共创营和社区项目，把点子做成真实产品
- 优先获得导师辅导、生态合作、商业机会等资源曝光
- 参与社区治理与提案讨论，共同决定社区的发展方向
- 根据实际贡献，享有周边福利、项目分成、作品推广等激励

### 7.3 角色路径（即将推出）

成为共创伙伴后，你可以根据兴趣和专长同时扮演多个角色：

**创造者 (Creator)：分享自己产品的伙伴**
- 适合正在做项目、有作品的伙伴
- 价值：项目孵化、资源对接、产品推广、社区影响力
- 参与方式：黑客松参赛、Demo Show、Build in Public

**贡献者 (Contributor)**
- 适合愿意参与活动组织、社区运营的伙伴
- 价值：组织能力成长、商业项目分成、专职转化机会
- 参与方式：成为志愿者、活动统筹、其他社区贡献（如协调资源、提供技术支持等）

**分享者 (Sharer)**
- 适合擅长分享知识、技能的伙伴
- 价值：个人品牌建设、课程合作、讲师报酬
- 参与方式：工作坊、技术讲座、内容共创

### 7.4 如何成为共创伙伴

**认证条件（以下任选其一）：**
1. 参加黑客松并交付 MVP
2. 做一次活动志愿者（签到、摄影、场地等）
3. 完成一次公开分享（Lightning Talk、工作坊）
4. 参与社区内容贡献（周刊文章、案例采访）

**完善创造者档案：**
- 个人信息、技能标签
- 作品/项目展示
- 合作意愿说明

**认证流程：**
1. 完成一次有意义的贡献
2. 填写创造者档案
3. 由伙伴关系负责人审核确认
4. 加入共创伙伴群，解锁全部权益

我们优先邀请在社区中做过志愿者、讲师、黑客松获奖或长期活跃的伙伴升级为共创伙伴。欢迎加入我们，一起多元参与、共同成长！
