这是举办一场标准周周黑客松活动所需的物料列表。请在活动前 3 天对照此表进行检查。

## 🎒 社区标准物料包 (复用)

如果你是第一次申请办活动，社区会邮寄给你一个标准物料包（Iron Kit）。活动结束后请妥善保管，供下次使用。

| 物料名称                      | 数量   | 用途                   | 备注                 |
| :---------------------------- | :----- | :--------------------- | :------------------- |
| **社区横幅 (Banner)**         | 1 条   | 合影背景、挂在显眼位置 | 尺寸通常为 3m x 1m   |
| **社区易拉宝**                | 1 个   | 展示社区、放在显眼位置 | 尺寸通常为 180 cm x 80cm   |
| **主办方工作牌 (挂绳+卡套)**  | 10 套  | 区分志愿者与参与者     | 蓝色挂绳，带 Logo    |
| **黑客松贴纸 (Logo Sticker)** | 50+ 张 | 分发给参与者贴电脑     | 随包附赠一部分       |
| **路演倒计时器**              | 1 个   | 演示环节控时           | 可用 iPad/网页版替代 |
| **指引路牌 (A3/A4)**          | 若干   | 贴在电梯口、门口指引   | 需自行打印张贴       |

## 🥤 易耗品与现场采购 (报销)

这部分物资通常需要主理人或后勤志愿者在当地采购，保留好发票，依据[报销流程]进行报销。

### 必选支持
| 物料名称             | 建议数量   | 用途                       |
| :------------------- | :--------- | :------------------------- |
| **饮用水 (瓶装)**    | 人数 x 1.5 | 保证现场基础饮水           |
| **马克笔 (黑色/粗)** | 5-10 支    | 写名牌、Workshop 讨论      |
| **签字笔/水性笔**    | 2-3 支     | 写奖状、临时补签           |
| **便利贴 (多色)**    | 5-10 本    | 头脑风暴、分组标记         |
| **A4 纸 (白)**       | 1 包       | 签到表、临时草稿、指引打印 |
| **垃圾袋 (大号)**    | 1 卷       | 活动结束后清场             |
| **U 盘/移动硬盘**    | 1-2 个     | 作品备份与离线播放         |
| **透明胶带/地贴**    | 1-2 卷     | 固定线缆与指引牌           |

### 可选加分项 (提升体验)
- [ ] **零食/茶歇**：小饼干、水果、糖果（由于预算有限，建议从简）。
- [ ] **小礼品/奖品**：低成本但可分享的小礼品（比如周边）/零食（比如乐事），增强颁奖氛围。
- [ ] **奖状/证书**：提升颁奖仪式感，便于合影。
- [ ] **组号牌/桌牌**：按组队人数上限配足，方便分组与计数。
- [ ] **签到桌布/桌旗**：提升签到仪式感，方便合影。
- [ ] **入场券（如有）**：签到发放，可作为投票券。
- [ ] **桌牌/作品名卡**：组队后填写作品名与成员，提升仪式感。
- [ ] **社区横幅**：合影更有识别度。
- [ ] **插排/拖线板**：如果场地插座较少，建议多准备 3-5 个长线插排（非常重要！）。
- [ ] **转接头 (HDMI/Type-C)**：用于讲师投屏，最好备 1-2 个通用扩展坞。
- [ ] **蓝牙音箱**：如果场地无音响，用于播放背景音乐。

## 📄 打印资料
- [ ] **活动签到表** (建议打 2 份，一份备用)
- [ ] **Wi-Fi 密码卡** (贴在桌上或投影显示)
- [ ] **入群二维码** (签到台展示)
- [ ] **签到二维码/表单一页** (含分组与邮箱字段)
- [ ] **投票二维码/作品提交二维码** (路演或提交环节展示)
- [ ] **交通指引** (地铁/打车/地标)
- [ ] **角色分工/流程图** (入场即可看到，减少混乱)
- [ ] **共学教程速查卡/共学二维码** (一页式，便于上手)

### 🎨 社区介绍传单/海报生成
- **方案 A：直接生成**：使用 即梦 / Lovart / Banana 等工具，访问对应网站抽卡生成即可。
- **方案 B：HTML 模板**：使用 `ai-design/` 下的 HTML 模板生成（例如 `ai-design/flyer_bold.html`、`ai-design/flyer_minimal.html`），替换文案后导出打印。
- **推荐提示词**（Gemini3 生成的 A4 活动海报）：
```text
生成一个吸引人的 xx 活动的 海报 HTML。A4 大小，突出要点，给出不同的风格。
```
