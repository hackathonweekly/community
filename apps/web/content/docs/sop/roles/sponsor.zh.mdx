**一句话目标**：确保权益兑现与接待体验，交付高质量露出素材。

## 现场执行清单（活动当天）

### 活动前
- [ ] 对照赞助协议列权益清单并完成物料确认。
- [ ] 与宣发/中控对齐 Logo 露出位置。

### 活动当天
- [ ] 现场接待与露出执行到位，确保品牌信息准确。

### 活动后
- [ ] 整理素材并完成结案回传与感谢。


## 注意事项与应急

### 关键标准
- Logo/品牌名准确无误是底线。
- 交付素材要可用、可转发。

### 执行细节（按需参考）

**活动前**
- 对照赞助协议列权益清单（Logo/宣讲/展位/奖品）。
- 确认物料寄送与收货清点。
- 与宣发/中控对齐 Logo 露出位置。

**活动当天**
- 展位与易拉宝提前摆放，奖品上台就位。
- 赞助商代表到场迎接并安排座位。
- 口播/宣讲环节核对品牌名。
- 提前沟通摄影重点拍 Logo 与颁奖合影。

**活动后**
- 整理照片、数据与回顾链接。
- 完成结案回传与感谢。

<Callout title="大型活动可拆分" type="info">
可拆为商务对接岗与现场接待/物料保障岗。
</Callout>

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **临时到访** | 迅速安排座位与简短介绍 |
| **露出错误/缺失** | 立即调整 PPT/海报或现场口头补偿 |
| **物料未到场** | 联系物流并准备电子展示替代 |
| **赞助方不满意** | 现场沟通诉求，承诺后续补救 |
