这里包含了周周黑客松活动中各个岗位的详细执行标准作业程序（SOP）。


<Callout title="导读" type="info">
建议每个志愿者在活动前 **24 小时** 完整阅读一遍所属岗位的 SOP。
</Callout>

## 🏃 核心保障组 (必选)
- [**主持人**](./mc) - 控场、氛围、连接参与者
- [**物料/场地**](./logistics/) - 物料准备、场地布置、清场复原
- [**签到接待**](./logistics/check-in) - 入场核销、人流引导、第一印象
- [**宣发推送**](./pr) - 海报、推文、现场直播

## 🛠️ 技术与内容组 (按需)
- [**现场中控**](./operator) - 投屏、音乐、灯光、路演管理、计时
- [**技术导师 Mentor**](./tech-support) - 教程演示、巡场辅导、节奏推进
- [**分享嘉宾（可选）**](./speaker) - 内容输出、预演、控时

## 🤝 统筹与外联 (管理)
- [**活动主理人/统筹**](./organizer) - 总统筹、决策、志愿者管理
- [**赞助商对接**](./sponsor) - 权益兑现、接待、拍照
- [**鼓励师/顾问**](./advisor) - 现场统筹、协调与兜底

## 📸 影像与周边
- [**摄影**](./photo) - 拍照、录像、素材输出
