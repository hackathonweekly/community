**一句话目标**：作为资深统筹在现场把关流程、协调人手、处理突发。

## 现场执行清单（活动当天）

### 活动前（T-7 ~ T-1）
- [ ] 与主理人过流程表并标出高风险节点，准备应急方案。
- [ ] 确认关键岗位与展示/投票规则到位。

### 活动当天
- [ ] 巡查补位并协调堵点，必要时调整流程。

### 致谢环节（17:55）
- [ ] 提供具体高光案例，支持主持人致谢。

### 活动后
- [ ] 输出复盘建议与改进点给主理人。


## 注意事项与应急

### 关键标准
- 只做关键协调，不抢主理人决策。
- 先给可执行方案，再解释原因。

### 执行细节（按需参考）

**活动前（T-7 ~ T-1）**
- 与主理人过一遍流程表，标出高风险节点（时间/人力/物料）。
- 准备 2-3 个应急方案（嘉宾迟到/设备问题/时间超时）。
- 确认关键岗位到位（主持人、中控、技术导师 Mentor、物料/场地）。
- 提前确认展示时长与投票规则，避免现场临时改动。

**活动当天**
- 全场巡查流程与人手缺口，及时补位。
- 发现堵点就地协调，必要时调整流程。
- 关键节点提醒主持人与主理人，保持节奏。
- 关注志愿者状态，安排轮换与补水。
- 观察观众疲劳度，必要时建议中场休息或压缩展示。

**致谢环节（17:55）**
- 提供具体高光案例，支持主持人致谢。

**活动后**
- 输出复盘建议与改进点给主理人。

<Callout title="大型活动可拆分" type="info">
可拆为流程巡场统筹与志愿者关怀支持两岗。
</Callout>

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **流程超时** | 砍掉低优先级环节或压缩时长 |
| **关键岗位缺席** | 立即调度替补或亲自顶上 |
| **现场冲突** | 先安抚情绪，拉到一旁沟通 |
| **主理人被缠住** | 代为处理现场小问题，减少干扰 |
