> 小型活动可由物料/场地兼任；50+ 人建议独立安排签到接待。

**一句话目标**：在 30 分钟内完成入场核销，让参与者感觉顺畅、被欢迎。

## 现场执行清单（活动当天）

### 13:30 - 14:00 | 到位准备
- [ ] 确认签到台、入群码/WiFi 码、签到表单与备用纸质表。
- [ ] 完成分工与动线安排，提醒早到参与者入群与注册。

### 14:00 - 14:30 | 签到高峰
- [ ] 快速核销与引导入座，确保全员入群。
- [ ] 引导填写主题/技能/组队需求，避免拥堵。

### 14:30 - 15:00 | 迟到接待
- [ ] 留人值守，引导迟到者静默入场并补扫。
- [ ] 汇总未入群名单并交给主理人/宣发跟进。

## 注意事项与应急

<Callout title="角色核心目标" type="info">
你是活动的第一印象，负责让参与者快速、顺畅入场。
</Callout>

### 执行细节（按需参考）

**13:30 - 14:00 | 到位准备**
- 与物料/场地确认签到台已就位。
- 签到台位置避开入口拥堵，预留排队动线。
- 入群码、WiFi 码张贴清晰可见。
- 确认签到表单字段（签到 + 分组 + 邮箱/联系方式），现场扫码测试。
- 准备纸质签到表与备用笔，防止系统故障。
- 准备组号牌/桌牌，按每组上限人数配足。
- 准备主题登记二维码/表格，并测试大屏/主题墙展示。
- 现场分工（建议 2 人：A 角核对名单，B 角发放姓名贴）。
- 提醒早到参与者先完成入群与工具注册，减少开场拥堵。

**14:00 - 14:30 | 签到高峰**
- 询问姓名/手机号后四位，快速核销。
- 递姓名贴与笔，提醒写大并贴在胸前。
- 引导填写创作主题/技能/是否找队友，主题实时上墙。
- 发放入场券（如有），说明使用规则。
- 引导入座与扫码进群/连 WiFi，未入群者现场补扫。
- 巡场提醒未签到者补扫，确保全员签到。
- 未报名者先安排等待区，开场后视空位放入。

**14:30 - 15:00 | 迟到接待**
- 至少留 1 人在签到台留守。
- 14:15 正式开场后，引导迟到者从后门或侧边静默入场。
- 开场后 PPT/大屏循环签到二维码，提醒迟到者补扫。
- 汇总未入群名单，交给主理人/宣发跟进。

### 工作职责
*   **高效核销**: 确保 50+ 人在 30 分钟内快速入场，不造成拥堵。
*   **第一印象**: 提供热情、专业的接待体验。
*   **人流分流**: 引导不同身份（参与者、嘉宾、观众）快速找到位置。

<Callout title="大型活动可拆分" type="info">
可拆为入口引导与签到核销两岗，缓解排队压力。
</Callout>

### 常见问题
| 问题 | 解决方案 |
| :--- | :--- |
| **签到系统崩溃** | 使用打印版签到表备份，手动核销 |
| **未报名空降** | 先请他在旁等候，确认已报名者都入场后，如有空位再安排。 |
| **找不到名字** | 询问是否使用了昵称？查看手机号后四位；如确认报名成功截图，可手动补录。 |
| **排队太长** | 立即增开一条通道（如使用纸质表同时核销）。 |
| **未入群** | 现场扫码入群，登记未入群名单并提醒补看共学文档 |
| **主题未登记** | 现场引导补填主题/技能，或在群内接龙补录 |

### 锦囊
1.  **笔要多**: 签到台至少准备 5-10 支粗头马克笔，笔最容易丢。
2.  **大声引导**: 高峰期人多嘈杂，要敢于大声引导：“报名的这边排队！写好名字的往里走！”
3.  **二次定位**: 开场前 2 小时单独发一次定位与指引，减少迷路。
4.  **组号牌配额**: 按组队人数上限配足组号牌，避免现场不足。
