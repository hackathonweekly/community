**一句话目标**：搞定场地、物料和复原，保障动线通畅与现场安全。

## 现场执行清单（活动当天）

### 活动前一天
- [ ] 核对 [物料清单](/docs/sop/manual/materials)，确认场地与到达指引。
- [ ] 准备签到表、WiFi 密码牌与指引物料。

### 12:00 - 13:30 | 场地布置（必须在 13:30 前完成）
- [ ] 完成指引牌、签到台与桌椅/排插布置，确保动线通畅。

### 14:00 - 17:30 | 机动保障
- [ ] 保障动线与物料补给，协助临时需求。

### 17:00 | 投票环节
- [ ] 发放/回收投票筹码并协助计票。

### 18:00 | 清场复原（必做）
- [ ] 清场复原并处理遗失物。


## 注意事项与应急

<Callout title="角色核心目标" type="info">
你是现场的“后勤中枢”，负责物料与场地保障，让活动有序、可靠、可交付。
</Callout>

物料/场地负责物料准备、场地布置和清场复原，并在活动中提供机动支持。

<Callout title="核心任务" type="warn">
活动结束后必须还原场地，确保社区信誉，方便下次借用。
</Callout>

### 执行细节（按需参考）

**活动前一天**
- 核对 [物料清单](/docs/sop/manual/materials)（签到表、姓名贴、笔、投票筹码、A4 纸、排插）。
- 确认场地地址与楼层，更新群公告的“如何到达”（地铁/打车/地标/电梯指引）。
- 如路径复杂，补充简短指引视频或图片动线，并安排楼层接应。
- 确认空调/门禁等开放时段，必要时调整活动时间。
- 准备/打印签到表与 WiFi 密码牌。
- 打印本场《活动执行手册》表（份数=志愿者人数），方便现场查看。
- 准备易拉宝/横幅与指引路牌，确认张贴点位。

**12:00 - 13:30 | 场地布置（必须在 13:30 前完成）**
- 张贴指引牌（楼下大堂/电梯口/转角）。
- 摆放易拉宝或横幅，强化入口识别。
- 签到台靠侧边摆放，避免入口拥堵。
- 桌椅按分组岛屿摆放，留出走动通道。
- 铺设排插到各岛屿，**胶带固定地面线缆**。
- 搭建签到台（签到表、笔、姓名贴、投票筹码）。
- WiFi 密码牌放在显眼处。

**14:00 - 17:30 | 机动保障**
- 入口动线保持通畅，及时移走阻塞物。
- 物料不足时补给（姓名贴、笔、投票筹码）。
- 协助主持人/签到处理临时物料需求。
- 如有零食/茶歇，及时补充并提醒参与者取用。

**17:00 | 投票环节**
- 发放/回收投票筹码。
- 协助计票，提前确认计票方式。

**18:00 | 清场复原（必做）**
- 桌椅归位，排插收纳。
- 垃圾打包带走（瓶子/纸屑）。
- 检查遗失物，拍照发群。

<Callout title="大型活动可拆分" type="info">
可拆为物料准备、场地布置、清场复原三岗，并单独安排 [签到接待](./check-in) 负责入场核销与人流引导。
</Callout>

### 常见问题与应急处理

| 问题 | 解决方案 |
|------|---------|
| **有人找不到场地** | 提前安排一位志愿者在楼下/地铁口接应，或通过微信/电话远程指引 |
| **指引不清/易拉宝缺失** | 立即补贴 A4 指引并安排人肉指路 |
| **物料不够（如姓名贴、笔）** | 向附近便利店应急采购，或向场地方借用 |
| **排插不够/跳闸** | 检查是否有大功率设备（如电热水壶），分散用电负荷 |
| **场地空调/灯光故障** | 立即联系场地方物业，同时告知主理人调整流程 |

### 物料/场地锦囊
1.  **动线优先**: 签到台与通道不要堵，入口保持清晰。
2.  **备用物料**: 多备几支粗头马克笔和胶带，现场消耗最快。
3.  **人肉指引**: 高峰期安排一位志愿者在楼下接人，体验最佳。
