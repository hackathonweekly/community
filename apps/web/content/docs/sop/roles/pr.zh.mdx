**一句话目标**：让更多人看到活动高光，沉淀社区影响力。

## 现场执行清单（活动当天）

### 活动发布期（T-7 ~ T-5）
- [ ] 产出海报与报名页，讲清做什么、谁适合、有什么收获。
- [ ] 明确活动玩法与参与须知，并完成多渠道分发。

### 预热期（T-3 ~ T-1）
- [ ] 社群倒计时与交通指引提醒，补发参与须知。
- [ ] 汇总报名名单与入群情况，对接摄影素材需求。

### 活动当天
- [ ] 现场快讯与关键数据记录，必要时开启直播。

### 活动后（24 小时内）
- [ ] 收齐素材并发布回顾，引导参与者发布作品。


## 注意事项与应急

### 关键标准
- 24 小时内完成回顾，趁热扩散。
- 数据与作品信息准确可核。
- 活动介绍必须清晰，避免“到了现场才知道要做什么”。

### 执行细节（按需参考）

**活动发布期（T-7 ~ T-5）**
- 产出海报（主题/时间/地点/报名码）。
- 完成推文/报名页，讲清做什么、谁适合、有什么收获。
- 明确活动玩法与参与须知（自带电脑/预习教程/工具注册/软件安装）。
- 多渠道分发（公众号/微信群/朋友圈/活动平台）。

**预热期（T-3 ~ T-1）**
- 社群倒计时与交通指引提醒（地铁/打车/地标/电梯）。
- 发送参与须知：提交格式/大小建议、展示时长提醒、预习教程链接。
- 汇总报名名单，未入群者逐一提醒或私信补拉。
- 准备现场素材需求清单，提前对接摄影。

**活动当天**
- 现场快讯：签到/开发/路演/合影各 1 组图。
- 有条件则开启直播并在群内同步链接。
- 记录关键数据（人数/作品数/亮点）。

**活动后（24 小时内）**
- 收齐照片、作品数据与链接。
- 引导参与者发布作品与话题标签（可配小礼品激励）。
- 发布活动回顾推文/朋友圈总结。

<Callout title="大型活动可拆分" type="info">
可拆为海报报名、现场快讯/直播、复盘回顾三岗。
</Callout>

### 常见问题与应急处理
| 问题 | 解决方案 |
| --- | --- |
| **照片延迟** | 先用手机素材应急，明确交付时间 |
| **审核不通过** | 准备替代文案与渠道 |
| **数据不全** | 现场同步记录关键数字 |
| **参与者未入群** | 现场补扫入群码，活动前逐一私信提醒 |
