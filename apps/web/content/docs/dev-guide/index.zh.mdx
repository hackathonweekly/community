## 技术栈速览

| 类别 | 技术 |
|------|------|
| 前端 | Next.js 16 + React + TypeScript + Tailwind CSS |
| UI 组件 | Shadcn/ui + Radix UI |
| 后端 API | Hono.js |
| 数据库 | PostgreSQL + Prisma ORM |
| 认证 | Better Auth（邮箱、GitHub、Google、微信） |
| 国际化 | next-intl |
| 文档 | Fumadocs (MDX) |
| 部署 | Zeabur + Docker |
| 代码质量 | Biome + Husky |
| 测试 | Playwright (E2E) |
