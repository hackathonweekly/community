# 根目录文档索引

本文件用于整理与梳理仓库根目录下的 Markdown 文档，方便快速定位与阅读。

## 推荐阅读顺序（新同学）

1. `README.md`：项目概览、快速开始、核心功能、常用命令。
2. `DEVELOPMENT.md`：开发规范与协作约定（提交规范、分支、测试等）。
3. `CONTRIBUTING.md`：贡献流程与协作指引。
4. `DOCKER.md`：本地/生产的 Docker 部署指南。
5. `TROUBLESHOOTING_UPLOADS.md`、`VERSION_DEBUG_GUIDE.md`：常见问题与调试排查。
